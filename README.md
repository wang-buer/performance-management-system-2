## 1、项目介绍

成绩管理系统2拥有两种角色

教师：成绩查询、学生管理、添加课程、添加成绩、成绩排序等

学生：查看自己成绩和个人信息


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql 5.x
- maven项目：否
- 硬件环境：Windows 或者 Mac OS
